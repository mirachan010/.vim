# 使い方win  
1. vimをインストールする  
<https://www.vim.org/download.php#pc>  
2. gitをインストールする  
<https://git-scm.com/downloads>  
3. このレポジトリを"\~/"でクローンする（\~/.vim/にができたらおｋ）  

>cmd(Git bash)上で  
>cd ~/  
>git clone https://github.com/mirachan010/.vim.git .vim  

4. win.batを起動する(\~/.vimrcができたらおｋ)  
5. vimを起動して待つ（ちょっと時間かかるかも）  
6. 楽しむ！！
